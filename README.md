## Learn-Swift 

Swift开发应用，整理一份每个点的练习

##### 一些关键词(来自搜索)

- CG：表示该类、常量、定义、函数、类型是来自CoreGraphics.frameworks，苹果在开发SDK的时候对资源的一种整理命名方法，一般用于渲染，画图等
- NS：数据类型，如NSObject、NSString，NS是OC中的
- UI：一般用于普通的视图和控制器
