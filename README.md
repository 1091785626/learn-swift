##### 一些关键词(来自搜索)

- CG：表示该类、常量、定义、函数、类型是来自CoreGraphics.frameworks，苹果在开发SDK的时候对资源的一种整理命名方法，一般用于渲染，画图等
- NS：Next Step，一般是指数据，如NSObject、NSString
- UI：一般用于普通的视图和控制器
